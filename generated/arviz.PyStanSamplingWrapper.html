arviz.PyStanSamplingWrapper¶

class arviz.PyStanSamplingWrapper(model, idata_orig=None, log_like_fun=None, sample_kwargs=None, idata_kwargs=None)[source]¶

PyStan sampling wrapper base class.

See the documentation on SamplingWrapper for a more detailed description. An example of PyStanSamplingWrapper usage can be found in the pystan_refitting.

Warning

Sampling wrappers are an experimental feature in a very early stage. Please use them with caution.
